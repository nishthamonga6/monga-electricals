# Monga Electricals Website

Built with Next.js 14 (App Router), Tailwind CSS, and Framer Motion animations.

## Scripts

- dev: start development server on port 1702
- build: build for production
- start: start production server on port 1702
- lint: run ESLint
- format: run Prettier

## Quick Start (Windows PowerShell)

```powershell
# from the project root
npm install
npm run dev
```

Then open http://localhost:1702 in your browser.

## Customize

- Update colors in `tailwind.config.ts`.
- Edit content in `app/(sections)/*` and `app/contact/page.tsx`.
- Replace `app/robots.txt` and `app/sitemap.xml` with your domain.
- Add a real favicon under `app/icon.png` or `app/favicon.ico`.
- Place the provided store photo in the `public` folder as `public/monga-store.jpg` to show it in the About section.

## Facebook

We embed your page via the official Facebook Page Plugin at `components/FacebookPage.tsx`.
